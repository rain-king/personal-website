1. My hardware

I use my Thinkpad T440p laptop for studying. My laptop was having problems with the trackpoint not working properly and trackpad stopping working after suspending in linux. So I manually built an older kernel where my hardware works better from kernel.org (Version 5.4).

I figured my custom kernel made firewalld fail. So I instead browsed the web, and found out an option for the Fedora's newest kernel that made my touchpad wake after suspension again.

I couldn't make images work on this website, until I figured out the SELinux context was preventing the server from properly displaying the images. One of the quirks of Fedora, I would probably not had this problem with Ubuntu or Debian.
